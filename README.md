# Star
    一个基于C++和Qt实现的在线视频播放器，分为服务端和客户端。主要功能：
    1.在线浏览并观看服务器上有的影视资源
    2.通过影视名称，演员或者导演的名字搜索影视资源
    3.查看观看历史
    4.显示用户的观看和收藏记录
    5.评论影视
